About

I’m a software generalist, meaning I am a full-stack developer and hands-on coding architect. I am strongest at backend distributed systems and data engineering. I’m enthusiastic about AI-assisted development, and good old clean, lean software development, agile delivery, and collaboration.

I am fascinated by software related to data and semantic processing, e.g. machine learning, computational linguistics, natural language processing, and information retrieval. This dovetails nicely with modern AI driven development.

Technical directions

Currently active: Java, Kotlin, Spring Boot, Python, Bash, TypeScript, Clojure, Datomic, Elasticsearch, Lucene, cloud deployment (Kubernetes, AWS, GCP), AI-assisted coding.

When I have time: Agentic systems (MCP etc), Go, Big Data, deeper AI, NLP, visualization, Scala, React and modern web application frameworks in general.

Farther out (interests): Haskell, F#, Julia, Rust, PureScript, quantum computing, semantic web and RDF.
